MCP-ANY-REST

连接万物：适用于任何 REST 后端的通用模型上下文协议 (MCP) 服务器

快速上手查看文档

通用兼容

不挑后端语言，不挑框架。只要是 REST API，我们都能连接。

极速接入

通过 ZML 声明式语言，极大降低接入成本，配置即开发。

安全可靠

企业级安全设计，完善的 Token 认证机制，保护您的核心数据。

生态开放

拥抱开源，完全遵循 MCP 标准，与 Cursor、Claude 等无缝集成。

项目愿景

在 AI 极速发展的今天，大语言模型 (LLM) 展现了惊人的能力，但如何让 LLM 安全、高效地连接到现有的海量业务系统（REST APIs）始终是一个挑战。

MCP-ANY-REST 应运而生。我们的使命是消除 LLM 与业务数据之间的隔阂。

我们不仅仅是在构建一个工具，更是在搭建一座桥梁——一座让 AI 能够理解并操作现实世界数据的桥梁。通过遵循 Model Context Protocol (MCP) 标准，我们让任何 RESTful 服务都能零成本接入 AI 生态。

核心价值

Write Once, Connect Anywhere

无需为每个 AI 模型重复开发接口。一次配置，处处运行。
