## 功能规划

| 功能模块 | 子功能 | 详细说明 | 支持情况 |
| :--- | :--- | :--- | :--- |
| **1. 可视化调试及测试工具** | **界面功能模块** | 日志输出与测试结果显示；模块及工具列表导航；请求与响应参数定义查看；测试用例定义查看与管理 | 规划中 |
| | **测试用例管理** | 支持用例的创建、编辑、删除及执行 | 规划中 |
| | **测试上下文编辑** | 全局变量与环境变量定义；测试数据定义与管理 | 规划中 |
| | **测试流控制** | 支持顺序执行、并行执行；支持条件分支与循环执行 | 规划中 |
| | **测试记录与审计** | 记录执行结果（Pass/Fail）与耗时；记录执行环境（OS、版本等） | 规划中 |
| | **交互体验** | 提供用户友好的界面，用于发送请求、查看响应结果、调试；支持批量测试 | 规划中 |
| **2. LLM 自动化生成** | **多模型支持** | 内置 GPT-4、Claude-3、通义千问等主流模型接口，可按场景切换 | 规划中 |
| | **需求理解增强** | 自动实体抽取、意图识别与约束补全，生成结构化 Prompt | 规划中 |
| | **代码质量保障** | 生成后自动语法校验，引入静态规则与动态沙箱双重检查 | 规划中 |
| | **交互式微调** | 侧边栏“需求 ↔ 代码”双向 Diff，支持自然语言追加指令生成增量补丁 | 规划中 |
| | **风格与模板定制** | 学习企业私有术语，支持个人“代码风格模板” | 规划中 |
| | **导入/导出** | 直接落库测试用例管理模块，支持导出 .zml/Postman/JMeter | 规划中 |
| | **持续学习闭环** | 用户反馈回流强化学习，支持个人数据集微调小模型 | 规划中 |
| | **可视化追踪** | 全链路版本回溯，生成图文报告展示趋势 | 规划中 |
| **3. 后端接入案例** | **代码托管平台** | **Gogs**：一键导入仓库、Webhook 管理、Issue & PR 测试 | 部分支持 |
| | **DevOps / CI/CD** | **Jenkins**：流水线状态轮询、构建产物下载、制品库清理验证 | 规划中 |
| | **企业协作工具** | **Zentao**：项目管理，任务分解 | 支持 |
| | **社区贡献** | 提供开放适配模板与脚手架 CLI；案例库实时展示下载量与性能基准 | 规划中 |
