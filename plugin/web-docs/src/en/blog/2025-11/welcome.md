# Welcome to MCP-ANY-REST

MCP-ANY-REST is a generic Model Context Protocol (MCP) server designed to simplify MCP integration for REST backends.

## Why Choose MCP-ANY-REST?

- **Universality**: Adapt to any REST API.
- **Efficiency**: Developed with Rust for superior performance.
- **Flexibility**: Declarative configuration using ZML language.
