# 欢迎使用 MCP-ANY-REST

MCP-ANY-REST 是一个通用的模型上下文协议 (MCP) 服务器，旨在简化 REST 后端的 MCP 集成。

## 为什么选择 MCP-ANY-REST？

- **通用性**: 适配任何 REST API。
- **高效**: 基于 Rust 开发，性能卓越。
- **灵活**: 使用 ZML 语言进行声明式配置。
